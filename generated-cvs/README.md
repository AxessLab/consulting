# Generated CV files

Assignment-specific CVs generated by the `generate` Slack command are stored
here as JSON, HTML, and PDF.

## Path structure

```text
generated-cvs/<assignment-id> - <assignment title>/<consultant-slug>-<language>.json
generated-cvs/<assignment-id> - <assignment title>/<consultant-slug>-<language>.html
generated-cvs/<assignment-id> - <assignment title>/<consultant-slug>-<language>.pdf
```

Example:

```text
generated-cvs/4 - Team Webbkonsulter - Roll 2 - Frontend Developer, Level 3 (Vinnova)/karin-toft-english.pdf
```

Folder names sanitize invalid path characters. A colon in the assignment title
becomes ` - ` for cross-platform compatibility.

## Review before sending

Generated CVs should be committed to the repository and reviewed by a human
before they are sent to a client. Link to the PDF in Slack replies.

## Rendering

Files are produced by `scripts/render-cv.py` from JSON that validates against
`schemas/cv-content.schema.json`. See `scripts/README.md` and
`automation-prompts/cv-generation.md`.
